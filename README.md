# Courtroom AI — Lie Detection System

A real-time AI system that detects deception using facial expressions and voice stress analysis.

## Features
- Facial micro-expression analysis using CNN
- Voice stress analysis using RNN
- Multi-modal fusion combining visual + audio signals
- Real-time credibility scoring (~88% accuracy)
- ~60% reduction in manual analysis time

## Tech Stack
Python, TensorFlow, OpenCV, Tkinter, librosa

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python gui_app.py`
